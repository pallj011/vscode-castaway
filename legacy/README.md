# Johnny Castaway - Modern Recreation

A cross-platform recreation of the classic 1992 Johnny Castaway screensaver.

## Features
- Classic and Modern graphics modes
- Holiday Easter eggs (Christmas, Halloween, St. Patrick's Day, etc.)
- Randomized events and animations
- Cross-platform support (Windows 11, macOS, Linux)

## Project Structure
- `index.html` - Main entry point
- `styles.css` - Styling
- `js/engine.js` - Core animation engine
- `js/scenes.js` - Scene definitions and logic
- `js/renderer.js` - Canvas rendering with graphics mode support
- `js/calendar.js` - Holiday detection and Easter eggs

## Running
Simply open `index.html` in a modern browser.
